Evidence summary (what this index is for)

Key design points (primary care)
  • This specialty bucket is for conditions where the main GP contribution is rapid triage + safe routing (often 999/ED/SDEC), not community work-up.
  • Useful pages here focus on: red flags, “must not miss” differentials, immediate actions, and the minimum handover data to avoid delays.
  • Where guidance depends on local pathways (SDEC availability, direct specialties, imaging access), this is flagged as Local agreement needed.

Emergency medicine / Trauma — specialty index (primary care)

Use this page when

  • You want quick links to acute “route-now” pathways, and a reminder of the core handover data to document before transfer.

Pathways

  • Head injury (adults) — assessment + when to refer (NICE NG232)

General referral letter / triage metrics (ED/ambulance)

  • Time + mechanism: exact time of injury; mechanism (incl. fall height/stairs/RTC); witness account if available.
  • Observations: HR, BP, RR, Temp, SpO2 on air, GCS (incl. best E/V/M if not 15), pupils.
  • Key symptoms since injury: LOC, amnesia, vomiting count, seizure, headache, confusion/behaviour change.
  • Neuro exam: focal deficit, gait/ataxia, speech, vision, limb power/sensation.
  • Antithrombotics/coagulopathy: warfarin/DOAC/heparin/LMWH; antiplatelets (excluding aspirin mono); known bleeding/clotting disorder; last dose if known.
  • PMH: previous neurosurgery; epilepsy; alcohol/drugs intoxication; safeguarding concerns.
  • Injuries elsewhere / C-spine pain.
