Evidence summary (what this index is for)

Key design points
  • Respiratory presentations are common and high-risk when hypoxia, sepsis, PE, ACS, or respiratory failure are in the differential.
  • For most pathways, the highest-yield primary-care actions are: document observations (incl. SpO2), identify red flags, treat early where indicated (e.g., antibiotics for likely CAP), and include the right referral triage metrics.
  • Antibiotic regimens should always be aligned to local antimicrobial guidance (NICE provides principles and example options, but local resistance/formulary differ).

General referral letter / triage metrics (respiratory)

  • Core obs: HR, BP, RR, Temp, SpO2 on air (and whether on oxygen), NEWS2 if used locally.
  • Symptoms: onset, pleuritic pain, sputum/haemoptysis, wheeze, exercise tolerance, confusion, syncope.
  • Comorbids/risk: COPD/asthma, immunosuppression, frailty, recent hospitalisation, aspiration risk, pregnancy, VTE risk factors.
  • Exam: work of breathing, chest findings, peripheral oedema, signs of DVT.
  • Tests done: CXR (if obtained), ECG (if chest pain/SOB), point-of-care bloods if available, viral testing if relevant.
  • Treatment already given: antibiotics (drug + start date/time), steroids/bronchodilators, oxygen.
  • Social: ability to cope at home, support, safeguarding concerns.
