Evidence / intent

  • This page is primarily a local signposting index (not a guideline). It links to the drafted pages that contain the underlying triage metrics.
  • Where suspected cancer criteria are mentioned, they are anchored to NICE NG12 (Suspected cancer: recognition and referral) but the operational route (who accepts/which clinic) is local.
  • Use this page to keep referral letters consistent and avoid missing the basics (pregnancy test, infection screen, exam findings).
Local (D&G): routing assumptions (from current notes)
  • Vulval symptom referrals: General Gynae clinic (not a separate vulval clinic unless locally agreed).
  • Infertility referrals: Fertility clinic under Gynae.
  • Semen analysis is required for infertility referral/triage.

Gynae referral quick index (primary care)

Before you refer (almost always)

  • Pregnancy test (document result + LMP) for anyone of reproductive potential with bleeding/pain.
  • Speculum + bimanual if symptoms allow (document: cervix appearance, discharge, tenderness, mass).
  • STI screen where relevant (chlamydia/gonorrhoea NAAT; add others per sexual health).
  • Document cervical screening status (up to date? last result?) when bleeding is a feature.
  • Include what the patient is worried about and what you told them re timescales/safety-netting.

Bleeding problems

  • Postmenopausal bleeding → PMB page (suspected cancer pathway logic + what to include).
  • Unscheduled bleeding on HRT → Bleeding on HRT (BMS 2024 thresholds; optimise first when low risk).
  • Heavy menstrual bleeding → HMB page (first-line TX, when to image, when to refer).
  • IMB / irregular bleeding (non‑pregnant) → Abnormal uterine bleeding (endometrial/cervical risk flags, baseline tests).
  • Postcoital bleeding → Postcoital bleeding (infection/ectropion vs suspicious cervix; referral metrics).

Pain / infection / acute presentations

  • Suspected PID (reasonable suspicion, pregnancy excluded) → PID page (empiric treatment + 72h review; admit triggers).
  • Pelvic pain (acute vs chronic; endometriosis framing; non‑gynae red flags) → Pelvic pain page.
  • Suspected ectopic pregnancy / acute abdomen → same-day ED/acute assessment (do not route as routine gynae).

Mass / cancer-triage (adnexal)

  • Adnexal mass / ovarian cyst → Ovarian cysts page (RCOG/NICE NG12-based triage, CA‑125 framing, when to refer urgently).

Vulval and cervical findings

  • Vulval itch/pain/skin change/lump → Vulval symptoms page (red flags + biopsy triggers). Route: General Gynae (per current note).
  • Symptoms suggestive of cervical cancer (e.g., persistent PCB and/or persistent unexplained discharge) → Symptoms suggestive of cervical cancer (PCB/discharge) — referral & colposcopy (urgent referral triggers; what to include).
  • Cervical polyp → Cervical polyps page.

Endocrine / fertility / pelvic floor

  • Infertility → Infertility page (include semen analysis; ovulation evidence; duration/risk factors). Route: Fertility clinic under Gynae.
  • PCOS → PCOS page (Rotterdam + exclusion; metabolic/endometrial risk; referral triggers).
  • Prolapse → Prolapse page.
  • Urinary incontinence (women) → Urinary incontinence page (NICE NG123-based; referral triggers to urogyn/urology as appropriate).

Referral letter minimum (copy/paste)

  • Problem + duration; current severity; impact on daily life; what has already been tried.
  • LMP + pregnancy test result (date).
  • Exam findings (speculum + bimanual) including cervix appearance and any mass/tenderness.
  • Relevant tests already done (STI NAAT, FBC/TSH/prolactin as relevant, smear status, CA‑125/US where relevant).
  • Comorbidities and anticoagulants; BMI; safeguarding concerns where relevant.
  • What the patient expects/wants (e.g., fertility, symptom control, definitive surgery).
