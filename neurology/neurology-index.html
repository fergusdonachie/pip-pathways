Evidence summary (what this index is for)

Key design points
  • In neurology, the highest-risk “misses” in primary care tend to be secondary headache (SAH/meningitis/raised ICP), stroke/TIA, spinal cord compression/cauda equina, and seizure/acute confusional states.
  • For many presentations, the highest-yield actions are: identify red flags, document observations + neuro exam, and use a clear urgency/routing rule (999/ED vs same-day acute assessment vs urgent outpatient).
  • Local routing varies (stroke hotline, SDEC, direct neuro access, imaging access). Each pathway includes a Local agreement needed checklist so publication is not blocked by local operational uncertainty.

Neurology — specialty index (primary care)

Use this page when

  • You want quick links to neurology pathways, and a reminder of the common referral-letter minimum data (neuro exam + onset timings).

Pathways

  • Headache red flags (SAH / meningitis / raised ICP / other secondary headache)
  • Suspected TIA / minor stroke (same-day vs rapid clinic; aspirin now)
  • First suspected seizure / seizure recurrence (triage + urgent 2‑week specialist assessment)

General referral letter / triage metrics (neurology)

  • Time course: exact time of onset (or last known well), progression, and current status (improving/fluctuating/worsening).
  • Observations: HR, BP, RR, Temp, SpO2, GCS (if relevant).
  • Focused neuro exam: speech/language, face/arm/leg power, sensation, coordination, gait, cranial nerves, visual fields; document any focal deficit.
  • Headache specifics: thunderclap vs gradual, fever/neck stiffness, papilloedema/visual symptoms, exertional/valsava trigger, head trauma, pregnancy/post-partum.
  • PMH/risk: anticoagulation/antiplatelets, immunosuppression, malignancy, recent infection, vascular risk factors.
  • Tests done: capillary glucose (if neuro symptoms), ECG (if TLoC/palpitations), urinalysis/pregnancy test if relevant; imaging if already arranged.
Local agreement needed (D&G)
  • Default routing for same-day neuro red flags: ED vs SDEC/acute medicine vs direct neurology contact.
  • Access route for urgent imaging (CT head / MRI brain) for red-flag headache without clear ED indication.
  • TIA/minor-stroke pathway: who to call, where patients go today, and what baseline tests primary care should perform before transfer.
