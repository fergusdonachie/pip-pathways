Evidence summary (triangulated)

Key “red flag” triggers (NICE CG150)
  • NICE CG150 recommends evaluating people with headache and considering further investigations and/or referral when there are red-flag features including: worsening headache with fever; sudden-onset headache reaching maximum intensity within 5 minutes; new neurological deficit; cognitive dysfunction; impaired consciousness; recent head trauma; cough/valsalva/sneeze-triggered; exercise-triggered; orthostatic headache; symptoms of GCA or acute narrow-angle glaucoma; or a substantial change in headache characteristics.

Headache red flags — triage (over 12s)

Initial assessment (do now)

  • Observations: HR, BP, RR, Temp, SpO2.
  • Focused neuro screen: speech, visual symptoms/fields, limb power/sensation, coordination/gait.
  • Look for: meningism (neck stiffness/photophobia), non‑blanching rash, papilloedema (if competent).
  • Key risks: anticoagulants/antiplatelets, immunosuppression, malignancy, pregnancy/post‑partum, recent head trauma.

Emergency (999 / ED now)

  • Thunderclap headache (sudden onset; maximal intensity within ~5 minutes) / “worst ever” → treat as possible SAH.
  • Headache + fever with any of: neck stiffness, altered mental state, non‑blanching rash, seizure, or sepsis/shock features → possible meningitis/meningococcal disease.
  • New focal neurological deficit or reduced consciousness.
  • Post‑traumatic headache with concerning features (especially if anticoagulated).
  • Acute eye pain with red eye + reduced vision/haloes (suspected acute angle‑closure glaucoma) → emergency ophthalmology/ED per local route.

Urgent same‑day assessment (local acute front door)

  • Worsening headache with fever (even without classic meningitis features).
  • Triggered by cough/valsalva/sneeze or triggered by exercise.
  • Orthostatic (posture‑dependent) headache (consider CSF leak) — urgent review if persistent/severe.
  • Progressively worsening new headache or a substantial change in established pattern.
  • Persistent vomiting without another clear cause (raised ICP concern).
  • New headache in people with immunosuppression or known malignancy.

Urgent specialist pathways (local access varies)

  • Suspected giant cell arteritis (usually age ≥50): new headache ± scalp tenderness/jaw claudication/visual symptoms/systemic features → start steroids per local pathway and arrange urgent assessment.
  • Possible brain/CNS tumour features (progressive headache with focal deficit, seizure, cognitive/personality change, papilloedema) → follow NG12 thresholds + local direct‑access imaging/referral route.

If no red flags (likely primary headache)

  • Make a positive diagnosis (migraine, tension‑type, cluster, medication‑overuse) using NICE CG150.
  • Do not refer for neuroimaging solely for reassurance in diagnosed primary headache (NICE CG150).
  • Safety‑net: return urgently if any red flag develops (fever, neuro deficit, thunderclap, drowsiness/confusion, persistent vomiting, new visual symptoms).

Referral‑letter minimum (suggested)

  • Onset: thunderclap vs gradual; time to peak; duration; progression; triggers (cough/valsalva/exertion/posture).
  • Associated symptoms: fever/neck stiffness/photophobia, vomiting, neuro symptoms, visual symptoms.
  • Obs + neuro exam; fundoscopy findings if done.
  • PMH/meds: anticoagulants/antiplatelets, immunosuppression, malignancy, pregnancy/post‑partum, head trauma.

Local agreement needed (D&G)

  • Where should red‑flag headache without instability go locally: ED vs SDEC/acute medicine vs direct neurology?
  • If SAH suspected in the community: confirm local “immediate ED” instruction and any pre‑alert expectations.
  • Direct access route for urgent CT head/MRI brain for suspected tumour/raised ICP features (if not via ED).
  • GCA pathway details: who to call, same‑day ESR/CRP access, and steroid initiation responsibility.
Safety net
If the person deteriorates, develops a new focal deficit, becomes confused/drowsy, develops fever/neck stiffness/rash, or the headache becomes thunderclap/worst ever: treat as emergency and arrange immediate transfer.
